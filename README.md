# Evergreen Therapy Website

A responsive and modern website for Evergreen Therapy, designed to provide information about therapy services, contact details, and more. Built with clean HTML, CSS, and JavaScript for optimal user experience across devices.

This repository contains the production website for Evergreen Family Therapy, created as a client project and documented here for portfolio and reference purposes.

The site was designed to feel calm, approachable, and trustworthy, supporting clients seeking therapy services through clear content structure, gentle visual hierarchy, and accessible design choices.


## Role
UI/UX Design & Front-End Development


## Scope & Focus
- Mobile-first, responsive website design
- Accessibility-conscious layout and typography
- Clear service discovery and contact pathways
- Minimal, distraction-free user flows

## Features

- **Responsive Design**: Optimized for desktop, tablet, and mobile devices
- **Multiple Pages**: Home, About, Services, and Contact sections
- **Interactive Elements**: JavaScript-powered features for enhanced user interaction
- **Clean UI**: Modern and accessible design with custom CSS styling
- **Fast Loading**: Lightweight static site with no dependencies

## Technologies Used

- **HTML5**: Semantic markup for structure
- **CSS3**: Custom stylesheets for layout and design
- **JavaScript**: Interactive functionality and DOM manipulation
- **Assets**: Images and media stored in the `assets/` directory
